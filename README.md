# Text-Based RPG Game

A console-based role-playing game developed in C++. Players can choose a character class, fight enemies, collect items, and save their progress.

## Features

- Character creation
- Turn-based combat system
- Inventory management
- Enemy encounters
- Save and load game functionality
- Object-oriented design

## Technologies Used

- C++
- OOP concepts (Inheritance, Polymorphism, Encapsulation)
- File handling using fstream

## How to Run

### Compile
```bash
g++ src/*.cpp -Iinclude -o rpg_game
